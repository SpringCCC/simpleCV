# simpleCV

本项目致力于提供简化版本的计算机视觉算法实现，所有项目均是理解论文+参考代码后再实现。

如果有推荐的便于初学者学习的库，也欢迎在issue中提出和补充。

本项目大部分内容是来源于Github，不会用做商业用途，如有侵权，请联系笔者删除。
